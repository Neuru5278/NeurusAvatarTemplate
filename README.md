# vrc-template-Avatar

This template sets up a VRChat Project

Package Name: com.neuru.template.nat
Package Version: 0.3.1

- "https://Neuru5278.github.io/nat"에 접속하여 VCC에 모든 리포지토리를 추가하세요.(작동하지 않으면 VCC버전이 2.1.0 이상이 아니라는 뜻이므로 Settings에서 업데이트를 확인하세요.)

- VCC에서 Settings - Files and Folders - Templates를 클릭하여 나오는 폴더에 압축을 해제하세요.(폴더가 열리지 않는다면 파일탐색기에서 "%localappdata%\VRChatCreatorCompanion"라는 경로를 들어가 "Templates" 폴더를 생성 후 여기에 압축을 해제하세요.)

- VCC에서 Create New Project를 클릭하고 NAT 0.3.1 템플릿을 선택 후 프로젝트를 생성하세요.(프로젝트 생성 소요시간 약 5분, 프로젝트 첫 오픈 소요시간 약 10분)
